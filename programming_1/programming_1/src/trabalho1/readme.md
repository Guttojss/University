# Objectives of the folder
## What is in this folder?
- On this folder there are 3 files, The first is the PDF file of all the content that was given to me for the assigment, the second one is the java file of the assigment and the third and final file is this readme.
## Why is there this folder?
- This folder was made for the first programming assigment, which consists of making a Small in memory database for Films and Series.
## What is the assigment and rules?
- Some of the rules of the assigment are :
  - I can only use content given in-class.
  - I can't use global variabels.
  - I can only use one (1) Class with various methods.
- Some evaluation methods : 
  - Data Verification and protection.
  - Working code/ program .
  - Code identation and organization.
  - program structure and correct ussage of comments.

## What was the information given to you for this project?
- I recived a PDF file with some context about the program, some ground rules and evaluation methods of the project. I was given this as context :
  - "My Media Manager is the first version of a simple multimedia management application (films and series). An item has, a Title (string), a type(film or series), a launch year(intenger), if it has been seen or not and a classification given by the user (between 1 and 10, 0 means "No Classification").
  The program must allow the user to create and edit the items, modify/give classification, print the list in different ways (filters and searches).
  The aplication must store all the information in arrays, assumming a max amount of items to be one-hundred (100). "  

## What features has the assigment?
- This in memory database has a Delete, Create, Edit and Modify features.
